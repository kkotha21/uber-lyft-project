# Uber/Lyft Price Prediction

This project predicts Uber and Lyft ride prices using machine learning.

## Models Used
- Linear Regression
- Random Forest

## Features
- Distance
- Cab type
- Source & destination
- Surge multiplier
- Temperature & rain

## Evaluation
Models are compared using:
- MAE
- RMSE
- R²

## How to Run
```bash
python3 Project.py
